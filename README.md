## vue-refresher

### Key Points

1) CDN: fastest way to use vue, use for small proj and testing
2) Create Vue: large scale projs
3) Key features: components, directives, data & methods
- reactive data binding and component-based architecture (dynamic & interactive)
- meta frameworks - nuxt.js(SSR) & gridsome(SSG)
- used for UI & SPAs
- enhanced UI/UX, modularity, ogranisation, re-usability & performance
4) why vue? - simple, approachable, incrementally adoptable, flexible, good performance & size, easily scalable, one of the fastest web framework, active community & rich ecosystem
5) Great Performance: efficient rendering mechanism using VDOM, core lib -> light weight -> faster loadtime & better runtime performance

### Ways to use

- Two main ways to use: Options & Composition API
- View CDN, View CLI, We frameworks
- Create View: uses vite (HMR, Rollup), out of the box ts, huge ecosystem of plugins!
